# Real Estate Price Analyzer

This project is for analyzing prices across UK by scraping the data from legitimate sites

Property pricing data is fragmented and difficult to interpret quickly. Buyers, renters,
and analysts need an easy way to collect listing data and extract pricing insights by
area, size, and property type.


Data Sources
- Zoopla
- rightmove

Scraping Methods
- Python Requests
- Python Beautifulsoup
- Python Selenium
- Also some handpicked manual to have right mix



How much Data?
- 300-500 listings
- Number of locations: 3–5
- Listings per location: 50–100

Rightmove area is in Sq m
Zoopla area is in Sq ft
